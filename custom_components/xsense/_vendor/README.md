This directory contains the reviewed `wheemer-python-xsense==0.1.0.dev2026070501`
wheel snapshot used by the Home Assistant integration.

It is bundled here so Home Assistant does not need to install a direct GitHub
wheel requirement during startup. Do not edit these vendored files directly.
Update the `wheemer-python-xsense` package in its own repo, cut a new release,
then replace this snapshot from that release wheel.
